# StringsModeration
A moderation expansion for Strings.

## Features
 - Chat Cooldown
 - Link Filtering
 - Word Filtering
 - Spam Filtering
 - Extensive Logging w/Strings

## Installation
Install StringsModeration.jar and add to your plugins folder. 
Configuration is in the folder `Strings` in `channels.yml` and `moderation.yml`
